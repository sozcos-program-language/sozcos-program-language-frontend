# VUE3

### [内置指令列表](https://cn.vuejs.org/api/built-in-directives.html)

### 修改 npm 镜像地址
```shell
npm config set registry https://registry.npm.taobao.org 
```

### 使用 vite 创建vue项目
```shell
npm init vite@latest <project name> -- --template vue
```

### 安装 Element Plus UI库
```shell
npm install element-plus --save
```
