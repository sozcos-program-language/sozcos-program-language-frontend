指令

指令是带有前缀 v- 的特殊属性，用于在模板中表达逻辑。
一个指令的任务是在其表达式的值变化时响应式地更新 DOM。
我是一个正常的a标签 我是一个vue绑定标签 vue 简写a标签

v-if: 条件渲染

你能看到我吗?

v-for: 列表渲染。

  • {{ item.value }}

v-model: 双向数据绑定。

View 的改变能实时让 Model 发生变化，而 Model 的变化也能实时更新 View

Message is: {{ message }}

v-on: 事件监听器。

指令的参数
