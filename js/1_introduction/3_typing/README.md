## 数字类型转换

虽说js会自动转换类型, 但是在计算中, 例如: "6" + 1, 这时候就成了拼接字符串, 结果 = 61, 而不是7;

因此在遇到需要计算的地方, 还是要显式转换类型;

| 值            | 转换后                                                                                                     |
|--------------|---------------------------------------------------------------------------------------------------------|
| undefined    | 	NaN                                                                                                    |
| null         | 0                                                                                                       |
| true / false | 1/0                                                                                                     |
| string       | 去掉首尾空白字符（空格、换行符 \n、制表符 \t 等）后的纯数字字符串中含有的数字。如果剩余字符串为空，则转换结果为 0。否则，将会从剩余字符串中“读取”数字。当类型转换出现 error 时返回 NaN。 |
